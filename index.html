Optimization Methods for Neural Network Quantum States

A Summer Project at TU München with Dr Irene López Gutiérrez

In this project, we implemented a novel optimization method for neural network ansätze for many-body quantum systems using the discrete flow of the implicit midpoint method applied to the Schrödinger equation. The method preserves the symplectic form of Hamiltonian dynamics and exhibits a performance similar to stochastic reconfiguration, but precludes the calculation of matrix (pseudo-)inverses.

Neutron Star Parameter Estimation using Machine Learning

A Term Project with Dr Kinjal Banerjee, Dr Tuhin Malik

In this project, we developed a neural network equation of state (EOS) model to predict neutron star properties using a multivariate Gaussian distribution of the nuclear matter parameters (NMPs) as the input. The neural network EOS was also capable of constraining the higher order parameters via Bayesian inference of the NMPs using the network predictions.
