At BITS, I was guided by Dr. Kinjal Banerjee on using neural network equations of state for constraining higher-order neutron star nuclear matter parameters. I dabbled a bit in theoretical physics research and worked with Dr. Raghunath Ratabole on applying the similarity renormalization group to an interacting quantum field theory in the Daubechies wavelet basis. I was briefly also a student researcher at the APP Centre for Artificial Intelligence Research guided by Dr. Tirtharaj Dash where we collaborated with Dr. Lovekesh Vig at TCS Research and Innovation Labs on learning hierarchical representations of images.

Optimization Methods for Neural Network Quantum States

A Summer Project at TU München with Dr Irene López Gutiérrez

In this project, we implemented a novel optimization method for neural network ansätze for many-body quantum systems using the discrete flow of the implicit midpoint method applied to the Schrödinger equation. The method preserves the symplectic form of Hamiltonian dynamics and exhibits a performance similar to stochastic reconfiguration, but precludes the calculation of matrix (pseudo-)inverses.

Neutron Star Parameter Estimation using Machine Learning

A Term Project with Dr Kinjal Banerjee, Dr Tuhin Malik
[report]

In this project, we developed a neural network equation of state (EOS) model to predict neutron star properties using a multivariate Gaussian distribution of the nuclear matter parameters (NMPs) as the input. The neural network EOS was also capable of constraining the higher order parameters via Bayesian inference of the NMPs using the network predictions.

iGEM WikiSync

A software package developed for the iGEM competition
[code] [documentation] [iGEM BITS Goa 2020]

iGEM WikiSync is a web framework-agnostic Python package that lets iGEM teams upload multimedia and code to iGEM servers and lets teams setup continous deployment for their wikis by seamlessly integrating with CI/CD tools like Travis. It also checks user code for broken internal links and shows warnings.

Disclaimer (March 2022): WikiSync has not been tested for iGEM's new website following their migration away from the Mediawiki platform.
